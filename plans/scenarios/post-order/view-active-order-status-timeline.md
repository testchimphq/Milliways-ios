## Prerequisites

- Customer has an active order in progress.

## Test Steps

1. Open the order tracking screen for the active order.
2. Observe current status and timeline milestones.
3. Refresh or revisit the screen after a status transition.

## Expected Behaviour

- Active order status is visible.
- Timeline reflects the current progress stage.
- Updated state is shown when order status changes.
